Assignment 1

a) Each website has a unique link in the form www.https://. The role of web servers is to hold and allow access to different websites. Web browsers are programs which help users access the World Wide Web through search engines as well as direct HTML links. The most significant functions of a web server is to store, process and deliver web pages to clients

b) The internet is older than the World Wide Web. The internet is a network of networks. The World Wide Web is a system used to access the internet.

c) Intranet is basically a smaller internet which is used for local or restricted communication networks. It is a private place where the company can share their data company information and computing resources between employees. The differences between the intranet and internet is that the intranet has multiple connections to the outside internet while being limited to interconnected networks in a company or an organisation while the internet can be accessed by anyone for individual use.

d) TCP or IP also known as the Transmission Control Protocol or Internet Protocol is a group of communication protocols used to connect device to the internet TCP is basically the communication between a device and internet which consists of four main layers. Http or Hypertext Transmission Protocol is used to transfer data across the web. Http supplies a web page from a web server to web browser.
